Implementing the Forward Process

The forward process of diffusion involves adding noise to a clean image. That is, given a clean image \(x_0\), we want to get a noised image \(x_t\). We can achieve this by computing \(x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1- \bar{\alpha}_t} \epsilon \), where \(\epsilon \sim N(0, 1)\) and \(\bar{\alpha}_t\) is a cumulative product hyperparameter. One Step Denoising

A more effective approach for denoising is through using a pretrained diffusion model. In this implementation, we use a UNet to estimate the noise in a noised image \(x_t\), then removing this noise to obtain \(x_0\). This is done using an inverse of the equation I provided above. Iterative Denoising

The true strength of diffusion models lie in iterative denoising. We can efficiently simulate running \(1000\) timesteps by denoisng at a reduced set of strided timesteps. For each strided timestep \(t\), we have the next timestep \(t'\), where the latter corresponds to a less noisy image. I leave out the computation for sake of aesthetics, but it's similar to a linear interpolation between signal and noise. Classifier Free Guidance

We can improve upon the previous diffusion results by using Classifier-Free Guidance. Whereas we previously only used a noise estimate \(\epsilon_c\) conditioned on a general text prompt, we now also use a noise estimate \(\epsilon_u\) "conditioned" on a null prompt. The latter is actually our unconditional noise estimate. We can compute a noise estimate \(\epsilon = \epsilon_u + \gamma (\epsilon_c - \epsilon_u\)), where \(\gamma\) is a CFG scale parameter.
